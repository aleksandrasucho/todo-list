# todo-list
## Project Overview

This is a simple, interactive To-Do List web app that allows users to add, edit, delete, and mark tasks as completed. The app is built with basic HTML, CSS, and JavaScript, making it a great beginner project for anyone learning web development.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Simple and clean design

## Installation

Clone the repository:
https://github.com/aleksandrasucho/todo-list.git

Open `index.html` in your browser to view the app.

## Usage

1. Enter a task in the input box.
2. Click the "Add Task" button to add it to the list.
3. Click on a task to mark it as completed.
4. Click the trash icon to delete a task.
